# Motivation

Mecha has long used a panel feature with a very minimalist appearance. At the time, the goal was simply to simplify data entry for users and, of course, to simplify maintenance of the existing panel features. Over time, the need for new features arose, such as the ability to sort the list. This would enable me to create extensions for managing navigation links. Another example is to have a file picker button next to a text input to browse files. However, the minimalist design of the current panel prevented me from enhancing its features without breaking the existing ones.

I want to create a CSS library with a class system similar to Bootstrap’s, but with as few class helpers as possible to avoid resembling Tailwind CSS. It has to have all the features of Bootstrap and other features so that it can be compared to the GNOME or KDE desktop environment, but for the web.
